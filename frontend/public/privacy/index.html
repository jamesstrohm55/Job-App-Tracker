Overview

HireTrackr helps users manage job applications, contacts, and related email activity.

Information collected

The app may collect Google profile information used for sign-in, job application records entered by users, contact information entered by users, and Gmail message metadata or content when a user explicitly connects Gmail.

How information is used

Data is used only to provide application features including authentication, job tracking, contact management, analytics, and job-related email workflows.

Google user data

If a user connects Gmail, Google user data is used solely to display, categorize, and manage job-related email inside the application. Google user data is not sold and is not used for advertising.

Data sharing

User data is not sold and is only shared with infrastructure required to operate the service, if applicable.

Data retention

Data is retained only as long as needed to provide the service or until the user requests deletion.

User rights

Users may request deletion of stored data and may revoke Google account access through Google account settings.
